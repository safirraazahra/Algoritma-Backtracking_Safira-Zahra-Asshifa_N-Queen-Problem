# Algoritma-Backtracking_Safira-Zahra-Asshifa_N-Queen-Problem

Repository ini berisi implementasi algoritma backtracking
untuk menyelesaikan N Queen Problem menggunakan Python.

## Deskripsi
N Queen Problem adalah permasalahan menempatkan N ratu
pada papan catur NxN sehingga tidak ada ratu yang saling menyerang.

Algoritma yang digunakan adalah Backtracking.

## Cara Menjalankan Program

Jalankan perintah berikut pada terminal:

python nqueen.py

## Output
Program akan menampilkan proses percobaan penempatan ratu
dan solusi papan catur.
